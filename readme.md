  <p>CLI program which compares two configuration files and shows a difference for following formats:</p>
  <ul>
    <li>JSON</li>
    <li>INI</li>
    <li>YML</li>
  </ul>
  <p>Possible formats for report generation:</p>
  <ul>
    <li>JSON</li>
    <li>Structured Text</li>
    <li>Plain Text</li>
  </ul>
  <h2>Installation</h2>
  <p>npm install -g gendiff-test</p>
